# Restaurant-Ratings-Customer-Behavior-Analysis-India
Project Overview
This project focuses on analyzing restaurant ratings and customer behavior across India using a complete data analytics workflow. The goal was to uncover actionable insights related to customer preferences, pricing, ratings, cuisines, locations, and restaurant performance to support data-driven business decisions in the food and hospitality industry.
The project integrates Python, MySQL, Excel, and Power BI to demonstrate strong skills in data cleaning, analysis, visualization, and dashboard storytelling.
